# STEP 11 - Admin Panel

## Tujuan
Membuat dashboard admin untuk mengelola seluruh sistem.

## Menu Admin
1. Dashboard
2. Users
3. Sellers
4. Jerseys
5. Auctions
6. Bids
7. Winners
8. Payments
9. Shipments
10. Certificates
11. Reports
12. Settings

## Statistik Dashboard
- Total user
- Total jersey
- Total auction live
- Total auction closed
- Total transaksi
- Total omzet
- Pembayaran pending
- Pengiriman pending

## Prompt Coding
Buatkan admin panel untuk website lelang jersey. Buat layout sidebar, topbar, dashboard statistik, tabel users, tabel jerseys, tabel auctions, tabel payments, tabel shipments, dan tabel certificates. Admin dapat edit status, verifikasi pembayaran, input resi, dan melihat laporan omzet.

## Acceptance Criteria
- Admin panel protected role admin/superadmin
- Dashboard statistik tampil
- Data tabel dengan search dan filter
- Admin bisa melakukan aksi penting
